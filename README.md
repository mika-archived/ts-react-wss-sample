# TypeScript + React Hooks + WebSocket Sample

Sample chat application using the following stacks:

* TypeScript
* React Hooks
* Styled Components
* WebSocket (Socket.io)


## Development

```bash
$ yarn run start
# or
$ yarn run start:client # http://localhost:3001
$ yarn run start:server # http://localhost:3002
```


## Tests

```bash
$ yarn run test
```
